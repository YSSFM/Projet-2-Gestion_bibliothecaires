# Projet-2-Gestion_bibliothecaires

Ce projet s’inscrit dans le cadre de ma formation en Intelligence Artificiel et des Technologies Émergentes à l’École Supérieur de Technologie et porte sur la mise en place d’un système destiner à digitaliser, automatiser et informatiser la gestion des bibliothèques. L’objectif principale est de remplacer les méthodes traditionnelles de gestion par une solution numérique fiable rapide et sécurisé. Le système développé repose sur l’utilisation de Java Desktop connecté à une base de données MySQL.
